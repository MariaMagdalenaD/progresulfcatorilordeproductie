2. Instructiuni de folosire a paginii Web

 

Aplicaţia a fost realizată folosind editorul HTML Microsoft FrontPage, stiluri CSS (Cascade Style Sheets) şi elemente de javascript, fiind optimizată pentru o rezoluţie minimă de 1024 x 768 pixeli şi browser-ul Internet Explorer. Pentru ca anumite elemente de javascript să fie funcţionale trebuie setată securitatea browser-ului astfel încât să fie permisă rularea scripturilor şi a obiectelor de tip ActiveX.

Site-ul ce prezintă acest referat este organizat pe mai multe pagini aferente elementelor de cuprins ale referatului. Fiecare pagină este construită pe baza unui tabel ce ocupă întreaga suprafaţă vizibilă a browser-ului pentru o rezoluţie minimă de 1024x768 şi ajunge la o suprafaţă de 90% ocupare din suprafata vizibila a browser-ului pentru rezolutia optima 1280 x 1024 din întreaga pagină vizibilă a browser-ului.

Tabelul ce stă la baza construcţiei fiecărei pagini este alcătuit din cinci linii si două coloane, gestionate astfel:

  • prima linie (se întinde pe toată lăţimea tabelului) conţine titlul referatului

  • a doua linie (se intinde pe toată lăţimea tabelului) conţine o imagine statică;

  • cea de a treia line (se intinde pe toată lăţimea tabelului) conţine două link-uri catre prima pagina a   documentului şi datele de contact ale autoarei acestui referat.

  • cea de a patra linie a tabelului este împărţită în două coloane în care în prima coloană  conţine cuprinsul referatului respectiv capitolele si subacapitolele acestuia sub forma link-urilor, iar cea de a doua coloană a tabelului cuprinde conţinutul elementului din cuprins selectat, din prima coloană.

  • ultima (se întinde pe toată lăţimea tabelului) conţine drepturile de copyright.

Pagina de început a site-ului este index.html, care, după accesare va afişa titlul referatului, autorul si profesorul coordonator. Pentru a intra propriu-zis in conţinutul referatului se va accesa link-ul START poziţionat in dreapta-jos. Prin accesarea acestui link se va deschide documentul Prima pagină.html ce conţine cuprinsul referatului. Ulterior, fiecare zonă a referatului poate fi parcursă prin accesarea link-urilor aferente cuprinsului.

Meniul ce conţine capitolele şi subcapitolele referatului, este un meniu javascript de tip dropdown, unde elementele de funcţionare ale meniului sunt definite în fisierul sdmenu.js iar elementele de design ale meniului în fişierul sdmenu.css.

Pagina de test este organizată din trei formulare, fiecare formular reprezentând cate un tip de test (A,B,C). In fiecare formular se va completa raspunsul iar la final se va face un punctaj total, fără posibilitatea intervenirii asupra testelor deja parcurse.
